**¿Qué es el deep learning?**

> Es el área del Machine Learning que se centra en el uso de redes neuronales para resolver problemas más complejos (cognitivos) como Computer Vision o generación de lenguaje

**¿Qué tipos/arquitecturas de redes neuronales conoces?**

> Hay muchas pero las que hemos visto en el master con las DNN (redes neuronales densas) y los Transformers

**¿Qué es un embeddings?**

> Es una representación vectorial de texto que se genera a partir de redes neuronales y "contiene" infromación semántica de la palabra

**¿Qué es un transformer?**

> Es un tipo de arquitectura de red neuronal que se usa para problemas de lenguaje como clasificación de texto (BERT) o generación de texto (GPT)

**¿Qué es hacer finetunning?**

> Es un proceso de entrenamiento que se hace sobre un modelo ya entrenado. Se hace cuando quieres "afinar" un modelo entrenado sobre un conjunto de datos concretos

**¿Qué librerías conoces?**

> Keras, Tensorflow y Pytorch (en clase solo hemos usado Keras). Para transformers concretamente HuggingFace.
