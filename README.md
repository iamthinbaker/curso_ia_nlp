# Curso de IA y NLP

Este repositorio contiene clases de IA aplicado al mundo de lenguaje computacional, con el objetivo de servir como una "guía básica" para entender los conceptos, en mi opinión, más generales.


**Disclaimer**

* **Nivel**: En el contenido de este curso se espera cierto manejo de programación en Python así como del uso de herramientas de Data como pandas o jupyter

* **Objetivo**: La idea es que sirva de guia práctica, no de recopilación exhaustiva. La idea es que quien quiera aprender conceptos como LLMs

* **Filosofía**

* **Contenido**: El contenido tiene un enfoque práctico y trata de enseñar conceptos tradicionales del NLP. Seguramente tras la llegada del GenAI, mucho del contenido que se muestra en este repo ya esté en desuso, pero conocerlo ayuda mejor a entender cómo funcionan los sistemas de IA actuales.
