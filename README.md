# 🤖 Curso de IA y NLP

Este repositorio contiene clases de **Inteligencia Artificial aplicadas al procesamiento del lenguaje natural (NLP)**, con el objetivo de servir como una guía práctica y estructurada para entender los conceptos más importantes del área.

La idea es construir una base sólida de NLP tradicional antes de llegar a los Large Language Models (LLMs), entendiendo el camino conceptual que ha llevado hasta los sistemas actuales.

<br>

## 📌 Disclaimer

Este curso pretende ser un **manual práctico**, no una recopilación teórica exhaustiva.

El objetivo es explicar los fundamentos del **Machine Learning aplicado al NLP**, cubriendo los conceptos clásicos que permiten entender mejor cómo funcionan los sistemas modernos de IA y GenAI.

### 🎯 Nivel requerido

Para seguir el curso se espera:

- Manejo básico de **Python**
- Experiencia con herramientas de datos como **Pandas**
- Uso de **Jupyter Notebook**
- Conocimientos básicos de álgebra y estadística

---

### 📚 Enfoque

- Contenido orientado a la práctica
- Explicación de conceptos tradicionales de NLP
- Desarrollo progresivo desde ML clásico hasta Deep Learning y LLMs

Aunque tras la llegada del GenAI parte del contenido pueda parecer menos utilizado en producción, conocer estos fundamentos es clave para comprender los sistemas actuales.

<br>
